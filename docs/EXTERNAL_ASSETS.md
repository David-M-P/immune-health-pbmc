# External Assets

No files from the imported TRIPSO working trees were deliberately excluded during initial repository setup. All ordinary files present in the imported working trees are tracked directly in this repository. The original nested Git metadata is preserved separately in a local timestamped backup and is not part of this repository.

Any future external model checkpoints or datasets should be documented here rather than committed automatically. Documentation must not fabricate URLs or asset locations and must not expose credentials or sensitive storage details.
